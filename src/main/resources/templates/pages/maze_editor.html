Maze Editor

How to Use the Maze Editor

  1. Click on a tile in the grid to cycle between:
    • Open tile (white)
    • Wall (black outline)
    • End tile (flag)
  2. The starting position of the player is always the top-left open tile.
  3. Click Test Maze to navigate the maze before saving.
  4. Mazes must have at least one path from start to end; impossible mazes cannot be saved.
  5. All saved mazes are stored on the server and can be played later.
